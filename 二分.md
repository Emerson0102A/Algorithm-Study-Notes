# 二分查找

### 整数二分

```cpp
int l = 0, r = 1e9;
while (l + 1 != r)
{
    int mid = (l + r) / 2;
    if(a[mid] >= x) r = mid;
    else l = mid;
}
cout << r << endl;
```

### 浮点二分

```cpp
double l = 0, r = 1e9, eps = 1e-6;
while(r - l >= eps)
{
    double mid = (l + r) / 2;
    if(f(mid) >= 0) r = mid;
    else l = mid;
    cout << r << endl;
}
```

### 二分答案

```cpp
bool check(int mid)
{
    bool res = true;
    //do something to check the authurity of mid
    return res;
}
int l = 0, r = 1e9;
while (l + 1 != r)
{
    int mid = (l + r) / 2;
    if(check(mid)) r = mid;
    else l = mid;
}
cout << r << endl;
```
